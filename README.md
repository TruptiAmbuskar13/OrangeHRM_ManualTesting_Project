# OrangeHRM_ManualTesting_Project

Project Overview :- 
This is a Manual Testing Project performed on the OrangeHRM Demo Website focusing on Login Page functionality. The project includes Project Details, Test Scenarios, Test Cases, Decision Table Testing.

Website:- https://opensource-demo.orangehrmlive.com/web/index.php/auth/login

Objective:- 
To validate the Login page using positive and negative scenarios
Apply Decision Table Testing technique
Execute test cases and analyze results

All testing documents are compiled in a single Excel file.
